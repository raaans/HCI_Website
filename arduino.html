Arduino workshops

In these workshops, we learn to program hardware (such as servo-motors, LED displays, etc) with the help of an Adruino Uno; a small board with a simple microcontroller on it.

Workshop 1

In the first workshop, there were 7 exercises. I'll show you four of the things I programmed in this challenge:

3C: Blinky blinky LEDs

First, we learned how to control LEDs: wether they were on or off and to control their brightness through digital pulse-width modulation. In this first example I programmed two LEDs, one of which blinks twice as fast as the other.

×
× int rLed = 10;
int gLed = 9;

void setup() {
  pinMode(rLed, OUTPUT);
  pinMode(gLed, OUTPUT);
}

void loop() {
  digitalWrite(rLed, HIGH);
  digitalWrite(gLed, HIGH);
  delay(500);

  digitalWrite(gLed, LOW);
  delay(500);

  digitalWrite(rLed, LOW);
  digitalWrite(gLed, HIGH);
  delay(500);

  digitalWrite(gLed, LOW);
  delay(500);
}

4C: Slow Fade

In this exercise, I made sure two LEDs slowly faded in and out. Where the green LED fades out, the red LED fades in, and vice-versa.

×
× int rLed = 10;
int gLed = 9;

void setup() {
 pinMode(rLed, OUTPUT);
 pinMode(gLed, OUTPUT);
}

void loop() {
 int x = 1;

 for (int i = 0; i > -1; i = i + x) {
  analogWrite(rLed, i);
  analogWrite(gLed, 255-i);

  if (i == 255) {
    x = -1;
  }
   delay(5);
 }
}

6C: This has potentio

Using a potentiometer, you can turn a knob to generate a number by measuring the resistance it has. Knob goes left, resistance goes down. Knob goes right, resistance goes up. You can of course use this to control the brightness of LEDs:

×
× int rLed = 10;
int gLed = 9;
float sensorValue = 0;
int sensorPin = A0;

void setup() {
 pinMode(rLed, OUTPUT);
 pinMode(gLed, OUTPUT);
 pinMode(sensorPin, INPUT);
 Serial.begin(9600);
}

void loop() {
 sensorValue = analogRead(sensorPin);
 Serial.println(sensorValue);
 sensorValue = (sensorValue/1023) * 255;
 analogWrite(rLed, sensorValue);
 analogWrite(gLed, 255-sensorValue);
}
